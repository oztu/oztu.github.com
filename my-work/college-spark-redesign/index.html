College Spark Washington Website Redesign

You can view the redesigned website live at http://www.collegespark.org/.

Technologies used:

  • PHP with Zend Framework
  • JavaScript with Dojo Toolkit
  • MySQL database
  • XHTML 1.0, CSS
  • Adobe Photoshop and Illustrator

Key features:

  • Created a new graphic style guide for site and communications, updated branding.
  • Developed a purpose-build CMS for staff to manage website with ease.
  • Developed a dynamic engine to generate web pages.
  • Created a integrated password-protected portal for staff and board of directors.
