College Spark's Intervention Explorer

The Intervention Explorer is project I designed and developed for College Spark to make a data set they had on Community College Interventions easier to consume and explore. It's live at: http://www.collegespark.org/page/79/Intervention+Explorer.

Some highlights:

  • The data is managed through a Google Docs spreadsheet, making it super easy for College Spark to keep the information up-to-date.
  • The page is self-sustained, using only javascript. I used jQuery with a few plug-ins (Cookies and Simple Modal).
  • The color palate is "Gamebookers" from ColourLovers, which skirts some of College Spark's primary colors (I tried using them, but they didn't work out).
