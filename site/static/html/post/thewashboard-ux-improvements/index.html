My Work for Washington Scholarship Coalition

I worked as an external developer and advisor for theWashBoard.org, a scholarship matching website serving more than 40,000 students in Washington State. You can also view a reference letter from the project manager of theWashBoard.org.

Technologies used:

  • C# with .NET Framework and NHibernate
  • JavaScript with jQuery
  • MS-SQL database
  • HTML 4.1 and CSS2
  • Adobe Photoshop, Illustrator, and InDesign

Key highlights:

  • Found, documented, and fixed bugs ranging from database query issues to stylesheet errors.
  • Implemented designs from Photoshop documents
  • Redesigned and streamlined most of the website's user interface.
  • Refactored bloated and/or inefficient code.
  • Provided advising on standards, performance, and security issues.
