College Spark Washington Online Data Management System

This is an internal tool at College Spark Washington.

Technologies used:

  • PHP with Zend Framework
  • JavaScript with Dojo Toolkit
  • MySQL database
  • XHTML 1.0, CSS

Key features:

  • Ability to quickly modify and create forms and profiles automatically based on data models
  • Browsing and searching through data objects or compound data objects.
  • Exporting objects or filtered sets of objects to spreadsheets
  • Super easy user management (with permissions) and backups.
  • Aggregated news feeds (similar to Facebook's newsfeed system).
